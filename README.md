# MyDefrag Script 
## Directorio de Epic Games
Reubica los archivos de Epic Games al inicio del disco duro.
## Pasos
1. Desactiva los archivos de paginación
2. Desfragmenta con myDefrag.exe
3. Activa los archivos de paginación
